# GameJam_BrachioTeam
この作品は2025 年 8 月に行われた企業合同ゲームジャムにて優秀賞を取った作品です。
本作品を他社の選考やポートフォリオに使用することは許可されています。 

企業合同ゲームジャムURL：
https://gamejam.marv.jp/202508/index.html

開発期間
3日

開発人数
4名（エンジニア4名）

開発環境
Windows 11 / Unity 2022.3.62f1

担当箇所
・タイトル画面UIの実装
・ゲーム成功時の演出実装
・頭・手足の円オブジェクトの操作処理実装

操作方法
頭・手足の操作：クリック＆ドラッグ

作品概要
2025年8月に開催された企業合同ゲームジャムにて制作した作品です。テーマである「社交ダンス」から「リフト」の動きに着目し、さらにアイスダンスの要素を組み合わせることでゲームとして表現しました。プレイヤーは頭や手足をドラッグ操作で動かし、指定されたポーズを再現することでゲームを進行します。

制作で意識したこと
3日間という限られた開発期間のため、簡潔なゲーム性を目指しました。
また、初めて遊ぶ人でもルールをすぐ理解できるよう、分かりやすいUIや説明を意識して制作しました。
短期間でも完成度の高い作品にするため、遊びの分かりやすさと操作のしやすさを重視しました。
